编写程序

安装编译环境以及编写第一个Python程序

  • ubuntu环境下安装Python编辑器IDLE

bash # 查看本机python版本 python --version # 安装IDLE编辑器 sudo apt install idle-python2.7

  • 使用IDLE编写第一个Python程序

  • 打开IDLE

  • File --> NEW File

  • 在新弹出的窗口输入Python语句

    python print('Welcome to Python')

  • File --> Save 保存成 .py文件

  • 运行方法一

  • Run --> Run Moudle

  • 会出现一个新的窗口返回结果

  • 运行方法二

  • python python welcome.py

编译源代码

在运行.py文件时会自动创建相应的.pyc文件，.pyc文件中包含目标代码，这个目标代码是一种Python专用的语言，计算机可以高效运行，并且Python程序是在虚拟机上运行的。

  • 输入字符串

python name = input('') # 函数input返回用户输入的字符串，name最终指向用户输入的字符串 # 如果想获得数字，需要自行将接受的字符串转换为数字类型，使用int()，float()都可以
